# Grama-Khata 📒

A Smart Digital Ledger Android App for small shopkeepers built with Kotlin and Gemini AI.

## Features
- 👥 Customer management
- 💰 Give Credit / Take Payment tracking
- 🤖 AI-powered WhatsApp reminders (Gemini AI)
- 📊 Daily report with analytics
- 💾 Offline-first with Room Database
- 📱 Clean Material Design UI

## Tech Stack
- **Language:** Kotlin
- **Database:** Room Database
- **Architecture:** MVVM + ViewModel
- **AI:** Google Gemini API
- **Communication:** WhatsApp Intent

## Screens
1. Home Screen — Customer list with balances
2. Customer Detail — Transaction history
3. Add Customer — New customer form
4. Daily Report — Analytics + AI insights
